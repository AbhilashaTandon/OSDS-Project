# Gaomon PD1560 Linux Driver

This is a Linux driver for the Gaomon PD1560 graphics tablet, created as a semester project for my graduate-level operating systems class. 

I did all of my development within a virtual machine running a Linux Mint 22 guest OS, so I can't promise this will work on your machine, or will run without crashing your kernel or deleting your file system. Please use at your own risk.

The driver is currently in development. Please come back later if you would like to use it.
